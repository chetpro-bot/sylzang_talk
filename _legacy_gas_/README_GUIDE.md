# 🚀 '약국 실장톡' 설치 및 알림 설정 가이드

제공해 드린 코드를 사용하여 구글 시트와 연동되는 웹앱을 만들고, 휴대폰으로 실시간 알림을 받는 방법입니다.

## 1. 구글 시트 및 앱스 스크립트 설정
1. 새로운 **구글 스프레드시트**를 생성합니다.
2. 상단 메뉴에서 `확장 프로그램` > `Apps Script`를 클릭합니다.
3. 왼쪽의 `Code.gs` 파일 내용을 제가 드린 `Code.gs` 코드로 교체합니다.
4. `+` 버튼을 눌러 `index.html` 파일을 만들고, 제가 드린 `index.html` 코드를 붙여넣습니다.
5. 상단 툴바에서 `setupSheet` 함수를 선택하고 **실행** 버튼을 눌러 시트의 헤더를 자동 생성합니다. (최초 1회 필수)

## 2. 휴대폰 알림 설정 (텔레그램 추천)
구글 앱스 스크립트 자체는 휴대폰에 직접 알림을 보낼 수 없지만, **텔레그램 봇**을 다리 삼아 무료로 실시간 푸시 알림을 받을 수 있습니다.

1. **봇 생성**: 텔레그램에서 `@BotFather`를 검색해 대화를 시작하고 `/newbot` 명령어로 봇을 만듭니다. 여기서 **HTTP API Token**을 받습니다.
2. **내 ID 확인**: 텔레그램에서 `@userinfobot`을 검색해 자신의 **Id (숫자)**를 확인합니다.
3. **코드 수정**: `Code.gs`의 하단 `sendTelegramNotification` 함수 내 `YOUR_BOT_TOKEN`과 `YOUR_CHAT_ID`를 위에서 받은 값으로 바꿉니다.

## 3. 웹앱 배포
1. Apps Script 우측 상단 `배포` > `새 배포`를 클릭합니다.
2. 유형 선택에서 **웹 앱**을 선택합니다.
3. **다음과 같이 실행**: `나(Me)`
4. **액세스할 수 있는 사용자**: `모든 사용자(Anyone)` (또는 조직 내 사용자)
5. `배포`를 누르면 생성되는 **웹 앱 URL**이 휴대폰이나 PC에서 접속할 주소입니다.

---
> [!IMPORTANT]
> 구글 시트에 데이터가 기록될 때마다 설정한 텔레그램으로 즉시 알림이 가게 됩니다. 무료이며 매우 안정적입니다!
